main.py - основной код бота, который использует фотографии из датасета и с помощью обученной модели анализирует отправленное ему фото рентгеновского снимка грудной клетки.

pneumonia_detected.ipynb - ноутбук в котором происходило обучение модели , 

checkpoint.pth - сохраненная и обученная модель которую использует телеграм-бот.

Формат файла с расширением .pth используется для хранения нейросетей, созданных в PyTorch. Этот формат обеспечивает сохранение параметров модели, включая веса и настройки слоев нейронной сети.
